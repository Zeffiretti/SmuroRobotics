# SmuroRobotics
robotics parameters for smuro

## 1. Installation

### 1.1 Clone and install all dependencies

```git clone --recursive https://github.com/Zeffiretti/SmuroRobotics.git```

## 2. Quick Start

### 2.1 Load module path:

`run loadpath.m`

### 2.2 Get parameters:

`run SmuroParameters.m`
